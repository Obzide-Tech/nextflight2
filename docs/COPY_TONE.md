# Copy & tone

NextFlight is an aviation-themed academy. Every surface borrows from cabin language
without overdoing it. Spanish-first; English copy mirrors the same metaphors.

## Voice attributes

- **Calm captain** — confident, low-volume, never shouting.
- **Curated** — fewer words, sharper imagery.
- **Personal** — second person ("tu cabina", "tu vuelo"), never corporate "we".
- **Premium** — language hints at first-class travel; never flexes price.

## Do / don't

- Do open features with a metaphor, then explain plainly.
- Do keep CTAs short: "Pagar con Apple Pay", "Confirmar solicitud".
- Don't use exclamation marks as filler. Reserve them for genuine moments.
- Don't say "haz clic" — say "toca" or just write the verb directly.
- Don't apologize in error states. State the constraint and the next step.

## Error patterns

- Constraint first, fix second.
  - "Saldo insuficiente. Reduce el monto solicitado."
  - "KYC requerido. Completa tu verificación antes del primer retiro."
- Never use raw API messages user-facing. Translate.

## Empty states

Never blank. Always one short line that frames the future. Example:

> Aún no has solicitado retiros. Cuando confirmes tu primera comisión, aparecerá aquí.

## Aviation moments worth keeping

- "Tu cabina te espera." — first login post-signup.
- "Despegue confirmado." — premium upgrade success.
- "Identidad confirmada, vuelo seguro." — KYC approval.
